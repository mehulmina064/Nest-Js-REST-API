## Installation

```bash
$ npm install
or
$ npm install - force  //if getting version issues

And then-
  npm install tslib //because of the version issues
  npm install cache-manager@4.0.0 --save  //because of the version issues
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
